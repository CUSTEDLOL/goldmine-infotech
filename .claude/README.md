# Claude Superpowers

These superpowers are a set of specialized skills and instructions that you (Claude) can reference to improve your performance on various tasks.

## How to use
When you are about to start a task, check the `.claude/skills` directory for any relevant skills. Some key superpowers include:
- `test-driven-development`: Instructions for a robust TDD workflow.
- `systematic-debugging`: A structured approach to finding and fixing bugs.
- `writing-plans`: Guidelines for creating clear and effective implementation plans.
- `subagent-driven-development`: Patterns for managing complex tasks using subagents.

Before performing any major task, browse the available skills in `.claude/skills/` and follow the `SKILL.md` file within the relevant skill folder.
